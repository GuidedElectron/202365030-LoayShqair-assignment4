# Portfolio Website

## Project Description

This is a responsive personal portfolio website built using **HTML, CSS,
and JavaScript**.

It includes:

-   An **About Me** section with profile image, personal quote, and
    introduction\
-   A **Projects** section highlighting selected academic work\
-   A **Contact** form with client-side interaction\
-   Responsive design for desktop, tablet, and mobile\
-   JavaScript-powered features (smooth scrolling, theme toggle,
    greeting message)

The project follows proper separation of concerns:

-   HTML → Structure\
-   CSS → Styling & Responsiveness\
-   JavaScript → Interactivity

------------------------------------------------------------------------

## Setup Instructions (How to Run Locally)

### Option 1 --- Open Directly

1.  Clone or download the repository.
2.  Navigate to the project folder.
3.  Open `index.html` in your browser.

------------------------------------------------------------------------

### Option 2 --- VS Code + Live Server (Recommended)

1.  Open the project folder in VS Code.
2.  Install the **Live Server** extension.
3.  Right-click `index.html`.
4.  Click **"Open with Live Server"**.

## Short Summary of AI Use

ChatGPT was the only AI tool used in this assignment.

-   Generating the initial HTML portfolio structure\
-   Creating responsive CSS using Flexbox, Grid, and media queries\
-   Implementing JavaScript features (smooth scrolling, theme toggle,
    greeting message, form handling)\

-   Providing UI/UX layout suggestions\
-   Assisting with documentation writing

use.
